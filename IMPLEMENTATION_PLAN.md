## Track 2: i18n (Internationalization)

### Stack

Qt's native translation pipeline: `lupdate` → `.ts` files (XML) → `lrelease` → `.qm` files (binary) → `QTranslator` at runtime.

---

### Step 1 — Audit & Wrap Strings

Scan all files in `src/gui/` and `src/core/` for user-visible strings not already wrapped in `tr()`.

**Wrap:** all UI labels, button text, dialog titles, status messages, tray tooltips, error dialogs.
**Leave as-is:** logger output, JSON keys, SQL queries, internal identifiers.

Files to audit:
- `src/gui/MainWindow.cpp`
- `src/gui/ProductDialog.cpp`
- `src/gui/SettingsDialog.cpp`
- `src/gui/AlertHistoryDialog.cpp`
- `src/gui/TrayIcon.cpp`
- `src/gui/Widgets.cpp`
- `src/core/AlertManager.cpp`

---

### Step 2 — i18n Directory

```
i18n/
├── pricebell_en.ts     — English (source / canonical)
├── pricebell_ar.ts     — Arabic (RTL)
└── pricebell_fr.ts     — French (starter example)
```

`.ts` files are generated by `lupdate` from source and committed to the repo. Translators edit them. `.qm` compiled files are **not** committed — generated at build time.

---

### Step 3 — CMakeLists.txt Changes

```cmake
find_package(Qt5 COMPONENTS LinguistTools REQUIRED)

set(TS_FILES
    i18n/pricebell_en.ts
    i18n/pricebell_ar.ts
    i18n/pricebell_fr.ts
)

qt5_create_translation(QM_FILES ${CMAKE_SOURCE_DIR} ${TS_FILES})
qt5_add_translation(QM_FILES ${TS_FILES})
```

Output `.qm` files into `${CMAKE_BINARY_DIR}/i18n/` or directly into `assets/`.

---

### Step 4 — resources.qrc

Add all compiled `.qm` files to the resource bundle so they are embedded in the binary:

```xml
<qresource prefix="/i18n">
    <file>../i18n/pricebell_en.qm</file>
    <file>../i18n/pricebell_ar.qm</file>
    <file>../i18n/pricebell_fr.qm</file>
</qresource>
```

---

### Step 5 — Runtime Loading (`main.cpp`)

Before creating the main window:

```cpp
QTranslator translator;
QString locale = loadLocaleFromSettings(); // e.g. "ar", "fr", "en"
translator.load(":/i18n/pricebell_" + locale);
app.installTranslator(&translator);
```

---

### Step 6 — Language Setting in SettingsDialog

- Add a `QComboBox` listing available languages (populated from embedded `.qm` resources)
- On selection change: persist the locale to the `settings` table in SQLite
- Two options for applying the change:
  - **Restart prompt** (simpler): show a `QMessageBox` telling the user to restart
  - **Live reload** (better UX): reinstall translator and call `retranslateUi()` on all open windows

---

### Step 7 — RTL Layout Support

For RTL locales (Arabic, Hebrew):

```cpp
if (QLocale(locale).textDirection() == Qt::RightToLeft)
    QApplication::setLayoutDirection(Qt::RightToLeft);
```

Call this before `MainWindow` is constructed.
